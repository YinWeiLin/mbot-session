# 湖南省2026年考试录用公务员公告

根据公务员法和《公务员录用规定》等法律法规有关规定，湖南省公务员局将组织实施湖南省2026年考试录用一级主任科员及以下和其他相当职级层次公务员工作。

## 一、招录计划和职位信息

2026年全省各级机关共计划招录公务员10198名。报考人员可通过红星网（http://www.hxw.gov.cn）、湖南人事考试网（http://www.hunanpea.com）或"红星云"微信公众号查阅具体的招录单位、职位、人数和报考资格条件。

## 二、报考条件

报考人员应当具备下列资格条件：

- 具有中华人民共和国国籍
- 拥护中华人民共和国宪法，拥护中国共产党领导和社会主义制度
- 年龄一般为18周岁以上、38周岁以下（1987年1月至2008年1月期间出生）；应届硕士、博士放宽到43周岁以下；面向村社区干部定向招录放宽至45周岁；人民警察职位一般30周岁以下；特警职位25周岁以下
- 具有良好的政治素质和道德品行
- 具有正常履行职责的身体条件和心理素质
- 具备符合招录职位要求的文化程度和工作能力，须持有国家承认的学历（学位）毕业证书

现役军人、在读普通高校非2026届毕业生、在职公务员不能报考。因犯罪受过刑事处罚、被开除党籍/公职、被列为失信联合惩戒对象等人员不得报考。

## 三、报考程序

**网上注册**：2026年1月28日9:00至2月4日17:00，登录湖南人事考试网注册。一个身份证号只能注册一次。

**网上报名与缴费**：2026年1月29日9:00至2月4日17:00，每人只能选择一个职位报名。低保人员、残疾人可申请减免考务费用（申请截止2月2日17:00）。

**打印准考证**：2026年3月9日9:00至3月13日17:00，登录湖南人事考试网自行下载打印。

## 四、笔试

笔试实行分级分类考试，各科目满分均为100分。

- 市级以上普通职位：行政职业能力测验 + 申论（省市卷）
- 县级普通职位、乡镇职位：行政职业能力测验 + 申论（县乡卷）
- 法院、检察院、综合行政执法职位：行政职业能力测验 + 申论（行政执法卷）
- 法官助理、检察官助理：行测 + 申论（行政执法卷）+ 法律专业知识
- 公安机关人民警察：行测 + 申论（行政执法卷）+ 公安专业科目
- 财政、金融、会计、审计类职位：额外加考财经专业知识

**笔试时间**：
- 行政职业能力测验：2026年3月14日上午 9:00-11:00
- 申论：2026年3月14日下午 14:00-16:30
- 法律/财经/公安专业科目：2026年3月15日上午 9:00-10:30或11:00

## 五、资格审查和体能测评

按招录职位计划1:2比例，按笔试成绩从高到低确定资格审查对象。

人民警察、乡镇专职人武干部职位须在面试前进行体能测评，体能测评只进行一次。

## 六、面试

面试实行分级分类考试，具体时间另行公告。面试对象为资格审查和体能测评合格人员。

## 七、成绩合成和排名

- 普通职位笔试成绩 = 行测×50% + 申论×50%
- 法官助理、检察官助理笔试成绩 = 行测×30% + 申论×30% + 法律专业知识×40%
- 财政金融会计审计类笔试成绩 = 行测×40% + 申论×30% + 财经专业知识×30%
- 公安人民警察笔试成绩 = 行测×40% + 申论×30% + 公安专业科目×30%
- 综合成绩 = 笔试成绩×60% + 面试成绩×40%

综合成绩相同的，按笔试成绩排名；笔试成绩相同的，依次按各科目原始分之和、行测、申论、专业科目成绩排名。

## 八、体检和考察

按招录职位计划1:1比例，按综合成绩从高到低确定体检对象。体检合格人员进入考察程序，考察突出政治标准。

## 九、公示与录用

拟录用人员在省、市州公务员考试录用官方网站公示，公示期为5个工作日，公示内容含姓名、性别、准考证号、毕业院校等。公示期满无问题者办理录用手续。

## 十、补充录用

体检结束后，对前期出现空缺的职位将面向社会统一进行补充录用。补充录用公告在红星网发布。

## 十一、特别提示

本次招考不出版也不指定考试辅导用书，不举办也不委托任何机构举办考试辅导培训班。各类违规违纪行为将受到严厉处理，情节严重者可五年或终身限制报考，构成犯罪的依法追究刑事责任。

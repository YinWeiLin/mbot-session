# 2026年江西省考试录用公务员公告（招5115人）

根据公务员法和《公务员录用规定》等法律法规，江西省公务员局决定组织实施全省2026年度考试录用公务员工作。

## 一、招考职位

具体招考职位和报考条件等信息请在微信公众号"江西组工微讯"和江西人事考试网（www.jxpta.com）查询。

## 二、日程安排

- 网上报名：2026年1月22日9:00至29日17:00
- 网上缴费：1月22日9:00至31日17:00
- 打印准考证：3月9日9:00至13日17:00
- 公共科目笔试：3月14日
- 专业科目笔试：3月15日
- 公布笔试成绩：4月8日
- 公布第一批面试人员名单和调剂公告：4月中旬
- 资格审查、面试：4月底前
- 体测、体检：5月中旬前
- 考察、公示：6月中旬前
- 录用审批、备案：6月底前

## 三、报考条件

**基本条件**：
- 具有中华人民共和国国籍
- 年龄一般为18周岁以上、38周岁以下（1987年1月至2008年1月期间出生）；博士研究生及定向招录乡镇事业单位工作人员职位放宽至43周岁
- 拥护中华人民共和国宪法，拥护中国共产党领导和社会主义制度
- 具有良好的政治素质和道德品行
- 具有正常履行职责的身体条件和心理素质
- 具有大学专科及以上文化程度

**特殊规定**：
- 设区市及以下公安机关、监狱、戒毒所人民警察职位年龄一般30周岁以下；特警职位25周岁以下
- 有基层工作经历要求的职位须具备相应经历
- 定向招录乡镇事业单位工作人员须连续在编在岗5年以上且近3年考核合格

**不得报考情形**：因犯罪受过刑事处罚、被开除党籍/公职、被列为失信被执行人、在职公务员、现役军人、在读非应届毕业生等。

## 四、报名步骤

**网上注册及报名**：2026年1月22日9:00至29日17:00，登录江西人事考试网，一个身份证号只能注册一次，每人只能报考一个职位。

**网上缴费**：照片审查通过后缴费，1月31日17:00前完成，未缴费视为放弃报名。最低生活保障家庭、建档立卡脱贫家庭可免收考试费和体检费。

**打印准考证**：2026年3月9日9:00至13日17:00，登录报名网站自行下载打印。

## 五、笔试

**公共科目**：
- 3月14日上午 9:00-11:00 行政职业能力测验
- 3月14日下午 14:00-16:30 申论

**专业科目**：
- 3月15日上午 9:00-11:00 财经管理专业笔试（财政、金融、会计、审计类职位）
- 3月15日上午 9:00-11:00 公安专业笔试（公安机关执法勤务职位）

## 六、面试

**面试比例**：计划录用数4人及以下面试比例3:1；5人按12人确定；6人及以上面试比例2:1。

**综合成绩合成**（总分400分）：
- 省市、县乡综合管理和行政执法一般职位 = 行测 + 申论 + 面试×200%
- 财政、金融、会计、审计类 = 行测 + 申论 + 财经管理专业笔试 + 面试
- 公安机关综合管理和警务技术职位 = 行测 + 申论 + 面试×200%
- 公安机关执法勤务职位 = 行测 + 申论 + 公安专业笔试 + 面试

## 七、体检（体测）

按综合成绩从高到低等额确定体检对象。公安类职位需先通过体能测评方能进入体检。体检按《公务员录用体检通用标准（试行）》执行，人民警察按特殊标准执行。对体检结论有疑问的可在7日内申请复检，复检只能进行1次。

## 八、考察、公示、录用

**考察**：体检合格人员进入考察，突出政治标准，政治上不合格实行一票否决。

**公示**：公示期为5个工作日，公布监督电话。

**录用**：公示期满后按规定程序办理审批或备案手续。

**服务年限**：新录用乡镇公务员在乡镇机关最低服务年限为5年（含试用期）。

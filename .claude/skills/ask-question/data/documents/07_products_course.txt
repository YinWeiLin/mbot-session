# WiLyn公考课程套餐与价格一览

WiLyn公考针对公务员考试全流程提供系列课程套餐，覆盖笔试、面试、专项提升等不同备考阶段，满足不同学员的备考需求。

## 笔试课程套餐

### 鲲鹏旗舰班

- **价格**：19800 元
- **课时**：60 天封闭集训
- **适合人群**：成公意向强烈、追求高端封闭管理的考生
- **包含内容**：
  - 行测全科精讲（言语、数量、判断、资料）
  - 申论精讲与批改（省市卷 / 县乡卷 / 行政执法卷）
  - 专项模考 + 月度模考 + 全真模考
  - 专属班主任全程督学
  - 封闭住宿（保障性酒店）
  - 报名选岗指导 + 历年分数线查核
- **协议保障**：不过笔试退差价

### 鲲鹏强化班

- **价格**：9800 元
- **课时**：30 天封闭集训
- **适合人群**：有一定基础、需要针对性强化的考生
- **包含内容**：
  - 行测 + 申论强化精讲
  - 专项刷题与错题分析
  - 全真模考 2 次
  - 班主任督学
  - 封闭住宿

### 密押冲刺班

- **价格**：3980 元
- **课时**：10 天线下集训
- **适合人群**：考前冲刺、查漏补缺的考生
- **包含内容**：
  - 高频考点精讲
  - 押题卷 3 套 + 解析
  - 申论批改 2 次
  - 在线答疑到考前

### 在线自学班

- **价格**：980 元
- **课时**：6 个月有效期
- **适合人群**：时间灵活、自律性强的考生
- **包含内容**：
  - 全科录播课（行测 + 申论）
  - 题库刷题系统（10000+ 题）
  - 3 次直播答疑课

## 面试课程套餐

### 蝶变旗舰营

- **价格**：小围考生先过关再缴费，大围考生 16800 元
- **课时**：85 天封闭集训
- **适合人群**：小围考生（优先）、大围逆风翻盘考生
- **包含内容**：
  - 专岗专训（县乡岗 / 执法岗 / 省市岗 / 烟草岗）
  - 四大思维体系精讲
  - 脱敏训练 + 礼仪训练 + 演讲训练 + 体能训练
  - 小班训练（8-10 人/组）
  - 仿真模考考场还原
  - 专属班主任 + 标准住宿 + 结课礼物
- **协议保障**：小围不过全退，含体检、含政审

### 尖子冲刺营

- **价格**：8800 元
- **课时**：15 天封闭集训
- **适合人群**：小围考生、分差领先 4 分以上的大围考生
- **包含内容**：
  - 精英院长级师资主讲
  - 小班制实战训练（20 人/班）
  - 专项磨题 + 全真模考
  - 封闭住宿
- **协议保障**：不过退费

### 密训冲刺营

- **价格**：大围 19800 元（不过仅收 3900 元成本），小围不过全退
- **课时**：9 天（4 天理论 + 5 天模拟）+ 考前加练
- **适合人群**：面试时间紧、需要快速提升的考生
- **包含内容**：
  - 分岗专训（10-12 人/组）
  - 讲练考督四阶段教学
  - 仿真模考考场还原 1 次
  - 校区加练到考前
- **协议保障**：不过关退费，含体检、含政审

### 封闭特训班

- **价格**：5800 元
- **课时**：7 天（3 天理论 + 4 天实战）
- **适合人群**：时间有限、需集中突破面试的考生
- **包含内容**：
  - 精英师资主讲
  - 分岗题本训练（10-12 人/组）
  - 封闭住宿
- **协议保障**：不过关退费，含体检、含政审
